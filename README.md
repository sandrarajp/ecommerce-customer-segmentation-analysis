E-Commerce Customer Segmentation & Sales Analytics

This project analyzes the Online Retail II dataset to uncover customer behavior patterns, operational performance, and sales trends. The analysis focuses on customer segmentation using RFM analysis, product performance evaluation, return rate analysis, and seasonal sales trend identification.

Objectives
Customer Segmentation using RFM Analysis
Sales Trend Analysis
Product Performance Evaluation
Return & Cancellation Analysis
Customer Retention Insights
Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
SQL
Key Insights
Identified high-value customer segments.
Analyzed revenue contribution using Pareto principles.
Discovered seasonal purchasing patterns.
Evaluated regional sales performance.
Highlighted operational inefficiencies through return analysis.
Business Value

The insights generated can support customer retention strategies, marketing campaigns, inventory planning, and operational decision-making.
